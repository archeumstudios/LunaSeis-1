# 0007 - Primary task and multiclass scope

Date: 2026-07-13

Status: accepted for pilot design; revisit after waveform availability audit.

Keep binary event detection as the primary task. Permit deep-moonquake versus natural-impact classification only as exploratory secondary work. Do not train a headline four-class model: the conservative A/B audit yields only 18 shallow moonquakes and eight artificial impacts. Never compensate by placing augmented copies or windows from the same event/family across splits.
