# Decision 0014: attach Nonshallow Batch 2 integrity outcomes

Date: 2026-07-13

## Decision

Attach Batch 2 event-station gap/ATT outcomes to the unified registry using the same rules as Batch 1. At event level, retain a candidate when any audited station is usable; retain all questionable and rejected station evidence. Do not change catalog labels or evaluation groups, and do not use amplitude, RMS, or edge occupancy as an acceptance gate.

## Evidence

All 3,270 products and 2,145,595,149 bytes passed exact-size and official NASA-MD5 verification. The audit covered 599 events and classified 586 usable, three questionable, and ten rejected within Batch 2. When combined with Batch 1 station evidence, the registry has 946 usable, three questionable, nine rejected-only, and 282 not-yet-covered nonshallow events.

## Limits

The gap and ATT thresholds are provisional. A rejected request is not a rejected physical event when another audited station supplies a usable request. The audit establishes raw-window integrity, not event-label correctness, signal morphology, model readiness, or performance.
