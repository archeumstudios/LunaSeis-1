# 0001 — Initial project scope

Date: 2026-07-12

Status: provisional pending authoritative archive audit.

Use Apollo 12, 14, 15, and 16 Passive Seismic Experiment data as the initial scope. Do not mix Apollo 17 Lunar Seismic Profiling Experiment data into the primary benchmark. This limits early domain complexity and keeps Phase 0 focused on trustworthy event-to-waveform alignment.
